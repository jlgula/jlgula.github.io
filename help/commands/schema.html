schema

NAME

schema - displays the ThingSet schema of a device

SYNOPSIS

schema type

DESCRIPTION

Displays the full ThingSet schema of a device type including its default value. The type must be one of the supported types as shown by the devices.

EXAMPLES

>> schema Light

"loadResistance":48.0,
"on":false,
"suspended":false,
"price":10.0,
"priceLimit":10.0,
"gridController":"",
"subscription":[],
"simulatedAttributes" {
    "simulator":"",
    "time":"",
    "deviceCurrent":0.0,
    "vBus":0.0,
    "circuitElements":[]
}
