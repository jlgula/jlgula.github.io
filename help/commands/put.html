NAME

put - replaces a value in a ThingSet data object

SYNOPSIS

put path value

DESCRIPTION

Sends the Put HTTP message to the device as a request to change one value in a ThingSet data object. The value must be a JSON object with no spaces permitted. The path can be a relative URL starting with the device name or an absolute URL starting with the scheme (always "sim:" in the web app).

The path can be shortened or elided completely using the attach command.

The current published ThingSet protocol does not implement Put but the same effect can be obtained with a Patch message with only one attribute. This implementation adds Put to the supported messages.

EXAMPLES

>> put Light1/on true
