NAME

load - read and execute a script file from the file system

SYNOPSIS

load fileName

DESCRIPTION

Due to limitations of the browser platform the load command does not work from the command line in the web app. Use the load button instead which opens a file dialog.

Currently the load command understands the following file formats:

  1. Command line script file. This is just a sequence of commands in a text file as if they had been read from standard input. Lines beginning with # are ignored and it is recommended but not required to use a shebang in the first line as in: "#!/usr/bin/env odg" to help identify the file.

  2. A circuit element JSON file. See the circuit help for details.
