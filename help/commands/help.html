help

NAME

help - displays a list of commands or more detail on a particular command

SYNOPSIS

help [command]

DESCRIPTION

List the commands supported by the command line shell.

EXAMPLES

>> help

add <deviceType> [<url>] - add a simulated device, default url is its name
attach [<url>] - connect to device to skip entering scheme and host in URLs
circuit [<name>...][-i <inputFile>] - display the results of a circuit analysis
delete [<url>] [-V] - delete from an array on device
devices - displays a list of known devices that can be added
exit [<exitCode>] - halt all devices and terminate the application
get <url> [-V] - retrieve a code from a device
help [<commandName>] - list all commands
list - list active devices
load [file...] - load and optionally execute file contents
log - display the log to standard output
patch <url> <JSONobject>  [-V]- update target with values from JSON object like: {"EnableCharging":false}
post <url> <code> [-V] - create new entity
put <url> <code> [-V] - replace entity code
remove <name> - remove the server with the name from list of active servers.
schedule <timeFromStart> <targetURL> <value> - schedule a put to the target at a time relative to the start of the simulation
schema <name> - display contents of schema
simulate [duration [tickInterval]] - run a simulation for the specified duration (h:m:s) or once if not specified
version - display the release version of the application
