NAME

simulate - runs a simulation

SYNOPSIS

simulate [count [interval]]

DESCRIPTION

Runs a simulation and displays the results in terms of the bus voltage and current through each device at the end of each tick interval. If no count is provided, a single tick is performed at the default interval of 1 second.

This behavior changes if events are scheduled via the schedule command. When events are scheduled, the simulation will run at least until the last scheduled event completes and possibly longer if the count and interval extend beyond the last event.

The tick interval can include hours, minutes, seconds and fractions of a second in the form:

hh:mm:ss.fff
