circuit

NAME

circuit - displays a circuit analysis

SYNOPSIS

circuit

DESCRIPTION

Displays the full circuit analysis of the current bus showing the voltage at each node and the current through each device including all the elements in the circuit elements property of the device. The nodes and elements are renamed to ensure uniqueness by assigning a prefix derived from the URL of the device.

EXAMPLES

    add PowerSupply
    add Light
    circuit
    
nodeVoltage         PowerSupply1$VoltageNode      48.0000             
nodeVoltage         PowerSupply1$Vbus             47.9990             
nodeVoltage         Light1$Vbus                   47.9970             
nodeVoltage         Vbus                          47.9980             
nodeVoltage         GND                           0.00000             
deviceCurrent       PowerSupply1$V1               -0.999985           
deviceCurrent       Light1$rLoad                  0.999938            
deviceCurrent       busResistor                   4.79980e-05         
deviceCurrent       PowerSupply1$R1               -0.999985           
deviceCurrent       deviceResistor$PowerSupply1   -0.999985           
deviceCurrent       deviceResistor$Light1         0.999938

This example shows the analysis of a simple bus with just a power supply and a load.

NOTES

The simulator will perform an analysis of an arbitrary circuit of resistors, voltage sources and current sources using the load command and selecting a JSON file with its contents in the right format. The following shows such a file:

{
    "$schema":"https://org.opendcgrid/schema/circuit/0.0.1",
    "data":{
        "elements":[
            {"type":"Resistor","designator":"R1","value":2.0,"pins":["Node1","GND"]},
            {"type":"Resistor","designator":"R2","value":4.0,"pins":["Node2","Node3"]},
            {"type":"Resistor","designator":"R3","value":8.0,"pins":["Node2","GND"]},
            {"type":"VoltageSource","designator":"V1","value":32.0,"pins":["Node2","Node1"]},
            {"type":"VoltageSource","designator":"V2","value":20.0,"pins":["Node3","GND"]}
        ]
    }
}
